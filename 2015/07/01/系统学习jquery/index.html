JQUERY

1. jQuery的美元符号$有什么作用？

2. jQuery 库中的 $() 是什么?

答案：$() 是 jQuery库中的工厂函数，其作用可以定义选择器获取页面元素、创建页面元素及将DOM对象转换为jQuery对象

3. $(document).ready() 是个什么函数？为什么要用它？编写方式?

当dom树完全加载时，jquery允许你执行代码，使用的最大好处的好处是它适用于所有浏览器，解决了跨浏览器的难题
    * $(document).ready(function(){});
    * $().ready(function(){});
    * $(function(){})

4. window.onload 事件和 jQuery ready 函数有何不同？

前者除了要等待dom树加载构建完成，还要等待图片，视频，音频在内的所有外部资源全部加载完毕。如果花费大量时间就会感觉得延迟
后者只需等待dom树加载完毕就能执行，无需等待图片，视频等外部资源，因此执行效率更快。
而且document.ready()还可以使用多次，但只能使用一次onload().

5. $(this) 和 this 关键字在 jQuery 中有何不同？

$(this)返回的是jq对象，你可以对它调用多个jq方法，比如text()等，而this代表当前元素，它是js关键词中的一个，表示上下文的当前dom元素，你不能对它调用jq方法。

6. 使用 CDN 加载 jQuery 库的主要优势是什么 ?

*加载速度一般都会比本地加载速度快
*节省服务器的带宽，减少服务器的压力
*CDN提供缓存服务，第一次加载之后，不需要再次加载

8. jQuery 中的方法链是什么？使用方法链有什么好处？

方法链是对一个方法返回的结果调用另一个方法，使代码简洁明了。同时由于只对dom进行了一轮查找，性能方面更加出色

9. jQuery中的.get()、[]和eq()方法的区别？

* get()和[]两者都是将jQuery对象转换成DOM对象
* get()方法是由jQuery提供的方法
* []是利用jQuery对象是类数组对象
* eq()方法返回的还是jQuery对象
